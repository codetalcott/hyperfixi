fx-i18n spike

Each button below is authored with localized fx-* attributes. The fx-i18n.js hook rewrites them to canonical English before hyperfixi-hx sees them, so the original processor needs zero awareness of the localized vocabulary. Open DevTools → Elements to watch the rewrite, and DevTools → Network to see the fetch fire.

Spanish (Latin script)

— vacío —
fx-acción → fx-action, fx-método → fx-method, fx-disparador="clic" → fx-trigger="click", ...

Japanese (CJK script)

— 空 —
fx-アクション → fx-action, fx-トリガー="クリック" → fx-trigger="click", ...

Arabic (RTL script)

— فارغ —
fx-إجراء → fx-action, fx-محفز="نقر" → fx-trigger="click", ...
