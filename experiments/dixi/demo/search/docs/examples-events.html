Event handling examples

Once modifier

Execute an event handler only one time:

<button _="on click.once add .clicked to me then put 'Clicked once!' into me">
  Click me (works only once)
</button>

Prevent default

Prevent form submission:

<form _="on submit.prevent log 'Form submission prevented'">
  <input type="text" name="username" />
  <button type="submit">Submit</button>
</form>

Prevent link navigation in favor of an inline interaction:

<a href="/page" _="on click.prevent toggle .expanded on #details">
  Toggle Details
</a>
<div id="details" style="display: none;">Additional details</div>

Stop propagation

Stop a click from bubbling to a parent handler:

<div _="on click add .parent-clicked to me">
  Parent div
  <button _="on click.stop add .button-clicked to me">
    Click me (won't trigger parent)
  </button>
</div>

Debounce and throttle

Debounce a search input by 300ms so each keystroke restarts the timer:

<input type="search" _="on input.debounce(300)
  fetch /search?q=${my.value} as html
  then put it into #results" />
<div id="results"></div>

Throttle scroll events so the handler fires at most every 100ms:

<div _="on scroll.throttle(100) add .scrolled to me">...</div>
