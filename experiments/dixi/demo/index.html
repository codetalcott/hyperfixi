dixi M2 demo

Four buttons. The English one uses canonical fx-* attributes (regression check). The Spanish, Japanese, and Arabic ones use localized attribute names that get rewritten on the fly by their respective external locale files. Open DevTools → Elements to watch the rewrite, and DevTools → Network to see the fetches.

English (canonical, no-op)

— empty —
No rewrite expected. dixi sees lang="en" and skips.

Spanish (Latin script)

— vacío —
fx-acción → fx-action
fx-disparador="clic" → fx-trigger="click"
fx-objetivo → fx-target
fx-intercambio → fx-swap

Japanese (CJK script)

— 空 —
fx-アクション → fx-action
fx-トリガー="クリック" → fx-trigger="click"
fx-ターゲット → fx-target
fx-スワップ → fx-swap

— فارغ —
