# Testing Scripts

Scripts for testing extracted patterns and command implementations.

## Available Scripts

| Script | Purpose |
|--------|---------|
| `test-all-patterns.mjs` | Run all pattern tests |
| `generate-pattern-tests.mjs` | Generate test files from patterns |
| `test-discovered-commands.mjs` | Test discovered command patterns |
| `test-edge-cases.mjs` | Test edge cases and corner cases |
| `test-loop-commands.mjs` | Test loop command patterns |

## Usage

```bash
# Run from project root
node scripts/testing/test-all-patterns.mjs

# Generate new tests
node scripts/testing/generate-pattern-tests.mjs
```
