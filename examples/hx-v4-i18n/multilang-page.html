Localized htmx-compat attributes

Three sections on one page, each in a different language. Each button uses the localized htmx attribute names from its lang= scope's vocab module. Devtools shows the author-time attribute names — the orchestrator translates them to canonical English under the hood. The mocked fetch returns the URL it was called with so you can confirm each lang scope wired its own attribute correctly.

Open devtools (F12) → Network tab. Each button click issues a GET to its authored URL (/api/usuarios, /api/ユーザー, etc.). The mocked fetch returns plain text so the output div fills with the response.
