Localized hx-live reactive counters

Three counters, three languages, one shared reactive variable ($global_count). Each counter binds to it via the localized form of hx-live — Spanish hx-en-vivo, Japanese hx-ライブ, Arabic hx-مباشر. Clicking any button updates the shared variable and all three counters re-render in lockstep.

What you should see

Clicking +1, −1, or reset updates the shared $global_count. All three counters — Spanish, Japanese, Arabic — re-render to the same number because they all subscribe to the same reactive variable via the localized hx-live form. Reactivity dep-tracking is language-agnostic; the vocab orchestrator just routes localized attribute names to the canonical hx-live path.
