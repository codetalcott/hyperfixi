Vite Plugin Multilingual Test

Testing automatic language detection and optimal semantic bundle selection across 21 supported languages.

How it works: The vite-plugin scans your HTML for hyperscript keywords in different languages. It then selects the smallest semantic bundle that covers all detected languages.
Expected Detection:
  • Japanese keywords: トグル, 追加, 削除
  • Spanish keywords: alternar, agregar, mostrar
  • Korean keywords: 토글, 추가
  • Russian keywords: переключить, добавить
  • Hindi keywords: टॉगल, जोड़ें
  • Thai keywords: สลับ, เพิ่ม

Expected bundle: all (covers all 21 languages)

Testing Different Configurations

Modify vite.config.js to test different scenarios:

// Test 1: Auto-detect (current)
hyperfixi({ semantic: 'auto', debug: true })

// Test 2: English only (smallest bundle)
hyperfixi({ semantic: 'en', debug: true })

// Test 3: Explicit languages
hyperfixi({ languages: ['en', 'ja', 'ru'], debug: true })

// Test 4: Regional bundles
hyperfixi({ region: 'western', debug: true })     // en, es, pt, fr, de, it
hyperfixi({ region: 'east-asian', debug: true })  // ja, zh, ko
hyperfixi({ region: 'slavic', debug: true })      // pl, ru, uk
hyperfixi({ region: 'south-asian', debug: true }) // hi, bn
hyperfixi({ region: 'priority', debug: true })    // 13 most common languages

// Test 5: Full multilingual with grammar transformation
hyperfixi({ semantic: true, grammar: true, debug: true })
