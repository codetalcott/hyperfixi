What's happening

  1. The full hyperfixi.js bundle loads (provides Runtime + installPlugin).
  2. A module script imports @hyperfixi/reactivity and installs it.
  3. Then hyperfixi-hx.js loads. Its scanner sees hx-live, finds the live feature registered, and emits a live ... end block.
  4. Each button mutates $count via a regular set command — the write fires the reactivity graph's notify hook, which re-runs the live body.
