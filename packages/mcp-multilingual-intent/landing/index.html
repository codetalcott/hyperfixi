Auto-detect across 8 domains

        

          The detect_domain tool classifies natural-language input to the correct domain
          without a hint. Domain-specific patterns (SQL has FROM/WHERE, BDD has given/when/then)
          are scored by confidence; the highest match wins.

How it works

        
LLM Agent                    siren-mcp                    Hono Siren API
(Claude, Cursor, etc.)         (bridge)                       (this server)
        |                          |                              |
        |  MCP tools/call          |                              |
        |  action__parse_...  ---->|                              |
        |                          |  POST /api/parse  ---------> |
        |                          |                              |---> @lokascript/framework
        |                          |                              |     DomainRegistry
        |                          |                              |     CrossDomainDispatcher
        |                          |                              |
        |                          |  <--- Siren JSON ----------- |
        |  <--- MCP result --------|                              |
        |                          |                              |
        |  (tools stay stable)     |  (stateless, no rebuild)     |

      


      
      

        
MCP tools exposed

        

          
            

              Tool
              What it does
              Required input
            

          
          
            

              action__parse_multilingual_intent
              Parse text → semantic AST (action + roles + confidence). Auto-detects domain.
              text
            

            

              action__detect_domain
              Classify input to one of 8 domains. Pure classification, no compilation.
              text
            

            

              action__compile_auto
              Detect + compile in one call. Returns SQL, Playwright test, JSX, etc.
              text
            

            

              action__get_pattern_examples
              RAG lookup — few-shot examples from the patterns database for LLM context.
              prompt
            

            

              action__list_supported_languages
              Enumerate all supported language codes.
              —
